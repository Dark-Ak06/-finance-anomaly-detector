How Isolation Forest Works
Isolation Forest — алгоритм машинного обучения для детекции аномалий. Основная идея:

🌲 Принцип: Аномальные точки легче "изолировать" в дереве решений — им нужно меньше разбиений.
📊 Признаки: amount, hour_of_day, day_of_week, category_encoded, amount_zscore, hour_sin, hour_cos
🎯 Anomaly Score: чем ближе к 1.0 — тем аномальнее транзакция. Порог: контаминация × max_score
⚡ Сложность: O(n log n) — работает в реальном времени даже на больших датасетах

Этот проект демонстрирует: feature engineering, unsupervised ML, real-time scoring.
